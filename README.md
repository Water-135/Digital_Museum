# 🎨 Digital Museum Management System

## 📌 Project Overview
The Digital Museum Management System is a web-based application designed to manage museum operations such as exhibitions, users, and learning activities.

This system provides different roles including Admin, Instructor, and Visitor to ensure efficient management and interaction within the platform.

---

## 👨‍🏫 My Role: Instructor Module
I was responsible for designing and developing the **Instructor functionalities** in this system.

### 🔧 Key Responsibilities:
- Manage and create educational content for museum visitors
- Upload and organize course materials (e.g. descriptions, media)
- Monitor user participation and engagement
- Update and maintain instructor profile

---

## ⚙️ Features Implemented
- Instructor Dashboard
- Add / Edit / Delete Learning Content
- View Visitor Participation
- Profile Management

---

## 🛠️ Technologies Used
- Frontend: HTML, CSS, ASP.NET Web Forms
- Backend: C#
- Database: Microsoft SQL Server
- Development Tool: Visual Studio
